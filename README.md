# OBD2 DTC Checker Widget v1.1

JavaScript埋め込み型のDTC診断ウィジェット（JSON内包版）

## 使い方

```html
<link rel="stylesheet" href="./css/obd2-checker.css">

<div id="obd2-dtc-checker"></div>

<script src="./js/obd2-checker.js"
  data-company-id="garage_001"
  data-cta-url="https://example.com/contact"
  data-cta-label="点検を相談する">
</script>
